【Minecraft】[1.20.1] 美化模組推薦 (fabric)

發表於 2023-09-24 更新於 2024-06-08 分類於 Game ， minecraft 閱讀次數： 文章字數： 783 所需閱讀時間 ≈ 3 分鐘

模組推薦 [1.20.1]

01. Chat Heads

—【聊天欄玩家頭像】

在訊息旁邊顯示玩家的頭像。

02. Visuality

—【更多粒子效果】

增加下雨水面漣漪、礦物鎧甲發光、生物受傷粒子等等的效果。

03. Not Enough Animations

—【更多的玩家動作】

增加手持物品、使用武器、進食、划船、騎馬、爬梯、等等人物動畫。

04. Eating Animation

—【更多的進食動畫】

增加進食時，食物慢慢消耗的新動畫。

05. 3d Skin Layers

—【3D皮膚層】

讓你的第2層皮膚有立體的效果。

06. Item Model Fix

—【修復物品模型隙縫】

修復各種原版Mincraft物品模型的細小錯誤。

07. Hat List

—【Tab清單顯示皮膚修復】

讓Tab名單中的第2層皮膚能夠正常顯示。

08. LambDynamicLights

—【Lambda的動態光源】

補充Fabric端所沒有的OptiFine功能。
讓你手持火把或光源時，能夠照亮周圍。
以及讓掉落的物品，或生物發光等等(視覺上)功能。

09. Continuity

—【連接材質】

補充Fabric端所沒有的OptiFine功能。
實現無縫玻璃、無縫書櫃、草皮過渡到其他方塊等等材質連接的功能。
模組自帶基本玻璃、書櫃、砂岩連接材質包。
如果有裝Sodium(鈉)模組的話，還需要裝Indium(銦)才可以使用。

10. CIT Resewn

—【自定義物品材質】

補充Fabric端所沒有的OptiFine功能。
讓更多的物品可以使用材質包改變圖案。

11. Entity Texture Features

—【自定義生物材質】

補充Fabric端所沒有的OptiFine功能。
讓材質包實現生物材質的發光、隨機等等的特殊效果。
玩家高級皮膚的效果。

12. Entity Model Features

—【自定義生物模型】

補充Fabric端所沒有的OptiFine功能。
讓材質包可以使用不同的生物模型。
需要與 Entity Texture Features 一起安裝。
非常推薦搭配 Fresh Animations 材質包使用。

材質包推薦

01. Fresh Animations

更生動的生物動畫
需求 Entity Texture Features、Entity Model Features

02. xali’s Enchanted Books

附魔書材質
需求 CIT Resewn

03. xali’s Potions

附藥水材質
需求 CIT Resewn

04. Vanilla Tweaks

Vanilla Tweaks 原版(香草)調整
可以自己選擇需要的功能，自定義自己想要的材質包功能。
主要在原版的基礎上做些微的調整，沒有太強烈的風格，很適合原版生存使用。
