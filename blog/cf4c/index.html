Basics:

This is a crafter that can handle up to two-material recipes. It uses a local divider clock for stability with random inputs.
Adjust the number of copper bulbs for different clock speeds.
Commonly used versions are 12gt and 24gt, but there is also a 4gt variant available.
By using 3 copper bulbs, you can achieve a 48gt version. See the thread for the recipe setup guide.

Pros:

  • Random input safe
  • SSI filtered
  • Prefilled: eliminates the chance of crafting the wrong item
  • Supports both single-material and two-material inputs

Cons:

  • Prefilling requires up to 13 stacks of items for each slice
  • Available clock speeds may not be optimal for all recipes

Setup Guide:

Input Speed:

  • Single Material: 8gt input 4 item, Ratio: 2gt/each
  • Two Material: 8gt input 2 item, Ratio: 4gt/each

Output Speed:

If the products is greater than 1, it can be directly multiplied by.

  • 4gt clock: 18k/h
  • 12gt clock: 6k/h
  • 24gt clock: 3k/h
  • 48gt clock: 1.5k/h
